Welcome to Loki!

Loki is a testing ground for ideas I'm working out with Node addons and Angular6.

In order to make Loki work, you'll need to:

    Build the addons using v11.2.0 of Node. You will probably need to update the include paths in the Visual Studio projects. Possibly update module_file_path in config.js Install (with npm install) packages both in the root and in the loki-site folder. Build the site from the loki-site folder with ng build Note that ng build will include the contents of the src/assets/images folder, so if you've generated screenshots in development mode, those will be included in the production build's dist folder. To avoid this, just clear the images from that folder before building. Start the server from the root with npm start. Open a browser tab to http://localhost:8888/.
